# 🍺 Bar do Tuíba - Website

Bem-vindo ao site oficial do **Bar do Tuíba**! Um espaço digital criado para apresentar nosso cardápio de bebidas e porções aos nossos clientes e amigos.

## 📝 Sobre o Projeto
Este é um projeto de desenvolvimento web focado em apresentar os produtos do bar de forma simples, intuitiva e funcional, permitindo que o cliente entre em contato diretamente conosco via WhatsApp para realizar pedidos.

### 🛠️ Tecnologias Utilizadas
O site foi desenvolvido com tecnologias essenciais para web:
* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização responsiva, layout flexível (Flexbox/Grid) e animações visuais.
* **JavaScript**: Interatividade para rolagem suave (smooth scroll) e observação de interseção para animações de entrada dos elementos.

## ✨ Funcionalidades
* **Menu de Navegação**: Acesso rápido às seções de Bebidas, Porções e Contato.
* **Cardápio Visual**: Cartões de produtos com imagens e botão de "Fale conosco" direcionando para o WhatsApp do bar.
* **Responsividade**: Layout adaptado para dispositivos móveis e desktops.
* **Interatividade**: Animações suaves de carregamento dos cards ao rolar a página.

## 🚀 Como acessar
Para visualizar o projeto, basta abrir o arquivo `index.html` em qualquer navegador moderno.
